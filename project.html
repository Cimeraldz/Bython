Hướng dẫn học Python

I. Các kiểu dữ liệu

Các kiểu dữ liệu trong Python giúp thuận tiện trong việc lập trình, gán kiểu dữ liệu và tránh các lỗi về kiểu dữ liệu.

1. Số

Là kiểu dữ liệu nhập vào số. Bao gồm số nguyên và số thực.

Số nguyên: int()

Số thực: float()

II. In 'print()'

2. Chuỗi

Là kiểu dữ liệu nhập vào các kí tự (Bao gồm tất cả các kí tự có thể nhập từ bàn phím).

Chuỗi: str()

1. In một số nguyên

n = int(input())
print(n)
Input: 1
Output: 1

2. In một số thực

n = float(input())
print(n)
Input: 3.2
Output: 3.2

3. In một chuỗi

n = input() hoặc n = str(input())
print(n)
Input: Hello
Output: Hello

III. Phép toán

Phép toán trong Python bao gồm các phép cộng, trừ, nhân, chia và nhiều phép toán khác.

1. Phép cộng

a = 5
b = 3
print(a + b)
Output: 8

2. Phép trừ

a = 5
b = 3
print(a - b)
Output: 2

3. Phép nhân

a = 5
b = 3
print(a * b)
Output: 15

4. Phép chia

a = 5
b = 3
print(a / b)
Output: 1.6666666666666667

5. Phép chia lấy dư

a = 5
b = 3
print(a % b)
Output: 2

6. Phép chia lấy phần nguyên

a = 5
b = 3
print(a // b)
Output: 1

7. Phép lũy thừa

a = 5
b = 3
print(a ** b)
Output: 125

IV. Boolean

Kiểu dữ liệu boolean chỉ có hai giá trị: True và False.

1. True và False

a = True
b = False
print(a, b)
Output: True False

V. So sánh

So sánh trong Python bao gồm các phép so sánh bằng, lớn hơn, nhỏ hơn, và nhiều phép so sánh khác.

1. So sánh bằng

a = 5
b = 3
print(a == b)
Output: False

2. So sánh lớn hơn

a = 5
b = 3
print(a > b)
Output: True

3. So sánh nhỏ hơn

a = 5
b = 3
print(a < b)
Output: False

4. So sánh lớn hơn hoặc bằng

a = 5
b = 3
print(a >= b)
Output: True

5. So sánh nhỏ hơn hoặc bằng

a = 5
b = 3
print(a <= b)
Output: False

6. So sánh khác

a = 5
b = 3
print(a != b)
Output: True

VI. Toán tử logic

Toán tử logic trong Python bao gồm and, or, not.

1. Toán tử and

a = True
b = False
print(a and b)
Output: False

2. Toán tử or

a = True
b = False
print(a or b)
Output: True

3. Toán tử not

a = True
print(not a)
Output: False

VII. Điều kiện

Điều kiện trong Python bao gồm if, else, và elif.

1. If

a = 5
if a > 3:
    print("a lớn hơn 3")
Output: a lớn hơn 3

2. If-Else

a = 2
if a > 3:
    print("a lớn hơn 3")
else:
    print("a không lớn hơn 3")
Output: a không lớn hơn 3

3. Elif

a = 3
if a > 3:
    print("a lớn hơn 3")
elif a == 3:
    print("a bằng 3")
else:
    print("a nhỏ hơn 3")
Output: a bằng 3

VIII. Vòng lặp

Vòng lặp trong Python bao gồm vòng lặp for và while.

1. Vòng lặp for

for i in range(5):
    print(i)
Output: 
0
1
2
3
4

2. Vòng lặp while

i = 0
while i < 5:
    print(i)
    i += 1
Output: 
0
1
2
3
4

3. Vòng lặp lồng nhau

for i in range(3):
    for j in range(2):
        print(i, j)
Output: 
0 0
0 1
1 0
1 1
2 0
2 1

4. Vòng lặp với điều kiện

for i in range(5):
    if i % 2 == 0:
        print(i)
Output: 
0
2
4

IX. Mảng

Mảng trong Python là một danh sách các phần tử có thể chứa các kiểu dữ liệu khác nhau.

1. Khởi tạo mảng

arr = [1, 2, 3, 4, 5]
print(arr)
Output: [1, 2, 3, 4, 5]

2. Truy cập phần tử

arr = [1, 2, 3, 4, 5]
print(arr[0])
Output: 1

3. Thêm phần tử vào mảng

arr = [1, 2, 3, 4, 5]
arr.append(6)
print(arr)
Output: [1, 2, 3, 4, 5, 6]

4. Xóa phần tử khỏi mảng

arr = [1, 2, 3, 4, 5]
arr.remove(3)
print(arr)
Output: [1, 2, 4, 5]

5. Độ dài của mảng

arr = [1, 2, 3, 4, 5]
print(len(arr))
Output: 5

6. Đảo ngược mảng

arr = [1, 2, 3, 4, 5]
arr.reverse()
print(arr)
Output: [5, 4, 3, 2, 1]

7. Sắp xếp mảng

arr = [3, 1, 4, 5, 2]
arr.sort()
print(arr)
Output: [1, 2, 3, 4, 5]

X. Slicing

Slicing là cách để lấy một phần của mảng hoặc chuỗi.

1. Slicing mảng

arr = [1, 2, 3, 4, 5]
print(arr[1:4])
Output: [2, 3, 4]

2. Slicing chuỗi

s = "Hello, World!"
print(s[7:12])
Output: World
